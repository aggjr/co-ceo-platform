DER Lógico Canônico (CORE + INVEST)

Visão Arquitetural com Snapshot Diário e Interface entre Módulos

Sua Dúvida: Onde está o controle do Módulo?

Você tocou num dos maiores debates de Arquitetura SaaS. A resposta é: O Controle está na Camada de Aplicação (A Catraca).

Fisicamente, tabelas como INVEST_ASSETS apontam direto para ORGANIZATIONS para garantir altíssima performance (evitar fazer 4 JOINs a cada consulta).
MAS, nenhuma linha de código do sistema acessa o bloco INVEST sem antes passar pela tabela CONTRACT_MODULES. Se o CNPJ do cliente não possuir o module_code = 'INVEST' ativo nessa tabela, o Chassi (Backend) bloqueia o acesso à parede do banco de dados. A tabela CONTRACT_MODULES é a verdadeira "Catraca" que liga o Core aos Módulos Opcionais.
